# Mahadi Hasan Portfolio

A responsive personal portfolio website built with React, Vite, and Tailwind CSS.

## Development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```

## Google Search Console

After deploying the site to `https://www.mahadihassan.info/`:

1. Add the domain property in Google Search Console.
2. Complete ownership verification using the DNS record or HTML meta tag Google provides.
3. Submit `https://www.mahadihassan.info/sitemap.xml`.
4. Request indexing for the homepage.

The project already includes canonical metadata, structured data, Open Graph tags,
`robots.txt`, and `sitemap.xml`.
